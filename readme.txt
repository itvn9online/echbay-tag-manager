== Installation ==
1. Upload `echbay-tag-manager` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Go to menu `Webgiare Plugins` menu and `Tag Manager` menu in WordPress.
